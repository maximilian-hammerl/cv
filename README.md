# Curriculum Vitae

This repository contains my curriculum vitae, built with [Typst](https://typst.app) – a modern, fast markup-based
typesetting system designed as an alternative to LaTeX.

## Structure

- `main.typ` - Main CV document
- `translations/` - Language-specific content (German and English)
- `icons/` - Icons used in the CV
- `compile.sh` - Build script

## Download

The latest versions of my CV are available in the [Releases](https://github.com/maximilian-hammerl/cv/releases) section
in both [**German**](https://github.com/maximilian-hammerl/cv/releases/latest/download/cv-german.pdf) and
[**English**](https://github.com/maximilian-hammerl/cv/releases/latest/download/cv-english.pdf).
